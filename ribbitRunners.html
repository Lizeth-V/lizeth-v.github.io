Ribbit Runners

Ribbit Runners is a basic 2D multiplayer platformer developed as the semester project for CECS 327 - Introduction to Networks and Distributed Computing. The game features a single level and a local multiplayer architecture, designed primarily to explore distributed systems in game development.

This project was also accompanied by a research paper detailing the implementation of the client-server architecture.
